Remindly - Smart Task Management System

📖 Description

Remindly is a Digital Assistant Reminder web application developed using Django. It helps users manage daily tasks, schedules, and important events efficiently. The system stores reminders and notifies users at the right time, ensuring that no important activity is missed. It is designed to improve productivity and simplify task management for students, professionals, and individuals.

🚀 Features

  • Set time-based reminders
  • Add, update, and delete tasks
  • User-friendly interface
  • Real-time task tracking
  • Notification alerts

🛠️ Technologies Used

  • Python
  • Django Framework
  • SQLite Database
  • HTML, CSS

⚙️ Installation

1. Clone the repository
2. Navigate to project folder
3. Run python manage.py runserver
4. Open browser at http://127.0.0.1:8000/

▶️ Usage

Users can add reminders by entering task name, date, and time. The system will notify users at the scheduled time. Tasks can be edited or deleted easily.
